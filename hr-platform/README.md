# AI 人才管理平台

## 项目简介

基于 AI 的企业级人才管理平台，提供三个档次的定制化解决方案：

- **标准版**：中小企业基础人事+招聘
- **进阶版**：中大型企业AI人力运营
- **旗舰版**：猎头专属AI寻访系统

## 技术栈

- **前端**: React 18 + TypeScript + Vite 5
- **UI框架**: Ant Design 5
- **图表**: ECharts
- **状态管理**: Zustand
- **路由**: React Router 6

## 快速开始

```bash
# 安装依赖
cd hr-platform
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 核心功能

### 标准版（中小企业）
- 简历自动结构化解析入库
- 员工档案统一管理
- 基础招聘职位发布管理
- 简易人才筛选、关键词搜索
- 基础人力数据统计报表
- 入职/转正/离职流程线上化

### 进阶版（中大型企业）
- 多渠道简历批量导入+智能去重
- 岗位-人才AI智能精准匹配
- 组织架构管理、编制管控
- 全员绩效数据多维分析
- 员工流失风险预警
- 人才储备池分类管理
- 中高层人才简易盘点
- 管理层人力数据驾驶舱

### 旗舰版（猎头公司）
- 客户企业岗位需求统一建档
- 寻访任务派发、跟进流程管控
- 面试记录、背调信息、offer进度全流程闭环
- 人才库去重合并、黑名单管理
- 寻访业绩数据统计、成单率分析
- 高端人才保密库独立隔离存储
- 深度人才评估（离职原因、竞业、口碑等）
- 多层级权限隔离
- 人才动态长期跟踪
- 海量自定义标签体系
- 库内人才智能盘活

- 岗位管理类型
- 面试管理类型
- 猎头专属类型
- 数据统计类型
- AI能力类型
- 权限控制类型
